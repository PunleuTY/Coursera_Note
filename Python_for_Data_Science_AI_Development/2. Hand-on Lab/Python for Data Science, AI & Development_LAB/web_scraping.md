# Introduction of Web Scraping

Web `Scraping` = Web `Harvesting` = Web `Data Extraction`

- `Example Scenario of WHY Web-Scraping`: When your work is to gather all data to find the popular movies of 2022 from a website like IMDB. Of course you can go one by one page and copy & paste the data , but it's too overwhelmed and time-consuming. That's where web scraping comes in to help you to extract the needed information from the webpage.

-> **Web Scraping** is the process of automatically extracting data from websites, web pages, and online documents by using specialized software tools.

- People used it various applications such as Data Analysis, Mining, Price Comparison, Content Aggregation, and more.

- This is like you are query the data from database, but the web scraping allows us to get the data that store in the web pages
